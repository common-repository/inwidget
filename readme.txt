Instagram Widget will display your insta-profile on your website.

== Description ==

Instagram Widget will display your insta-profile on your website. It does not require you to provide your login details or sign in via oAuth, just need to put a Instagram username.

For display &quot;We are on Instagram&quot; using shortcode or installation widget in WordPress `Appearance > Widgets`.

== Installation ==

Just install from your WordPress "Plugins > Add New" screen and all will be well. Manual installation is very straightforward as well:

1. Upload the `inwidget` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it!

== Shortcodes ==

[instagram_widget user_name=&quot;instagram&quot; width=&quot;320&quot; inline=&quot;3&quot; title=&quot;My profile&quot; /]

user_name - Instagram User Name
width - Width iframe Wind (px) / default 320
height - height iframe Wind (px) / default 320
inline - Count Images in Line / default 3
title - Title Widget
